Facebook bejegyzés mellékelete

Programozási nyelv:  Python  (>3.7)

Telepítés:  
- előfeltétel:  python 3.7 (vagy felette)

    scipy.interpolate, statsmodels.tsa.stattools, sklearn, locale, time, os, itertools, string
